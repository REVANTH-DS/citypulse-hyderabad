# CityPulse Hyderabad

CityPulse Hyderabad is an urban liveability analytics and decision-support platform designed to help users compare selected localities in Hyderabad using environmental and urban-service accessibility indicators.

## Problem

Information about air quality, healthcare access, educational services, public transport, and essential urban services is often available across different data sources.

Comparing localities using these factors can be difficult because the data is fragmented and the indicators use different measurements.

## Solution

CityPulse Hyderabad collects, processes, and analyses urban data to create comparable locality-level indicators.

The platform will allow users to:

- Explore selected Hyderabad localities
- Compare two localities
- Analyse environmental indicators
- Analyse urban-service accessibility
- Adjust indicator importance
- View a weighted CityPulse Score
